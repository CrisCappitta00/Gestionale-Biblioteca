\# Gestionale Biblioteca in Java

Un piccolo progetto educativo per ripassare i concetti di \*\*OOP\*\* (Object Oriented Programming).



\## Caratteristiche

\- Gerarchia di oggetti (Libri, DVD, Riviste).

\- Gestione dei prestiti tramite \*\*Interfacce\*\*.

\- Sistema di eccezioni personalizzate.

\- Controllo dei limiti di prestito tramite una classe Utility.
