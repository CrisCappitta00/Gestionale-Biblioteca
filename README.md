\# Gestionale Biblioteca Java



Un progetto didattico sviluppato per ripassare i pilastri della programmazione a oggetti (OOP) in Java.



\## Struttura del Progetto

\- \*\*Model\*\*: Gerarchia di oggetti (Libri, DVD, Magazine) con classi astratte e interfacce.

\- \*\*Logic\*\*: Manager per la gestione del catalogo e dei prestiti tramite HashMap e ArrayList.

\- \*\*UI\*\*: Main con integrazione del menù e della logica di comunicazione con il client.

\- \*\*Exceptions\*\*: Gestione robusta degli errori con eccezioni personalizzate.

\- \*\*Utility\*\*: Validazione dei limiti di prestito.



\## Tecniche Utilizzate

\- Ereditarietà e Polimorfismo.

\- Interfacce per comportamenti selettivi.

\- Gestione delle Collections (List, Map).

\- Versionamento con Git (Main/Develop workflow).
